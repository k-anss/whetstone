# 禁区 — 见即删

不必跑测试。这些从不存活。

## 1. 情绪符号

- 专业/留档文案中的任何 emoji
- ✨ 🎉 类装饰字符
- 感叹号堆叠(!!、!!!)

## 2. 感谢 / 引流话术

整句删:

- "感谢您耐心阅读本文,如有收获,欢迎分享"
- "如果觉得有用,请转发给身边的朋友"
- "欢迎点赞、收藏、关注"

## 3. 作者吹捧

留名,删头衔堆砌:

- "本文作者X,资深独立研究者,长期深耕该领域多年" → "X"

真实的许可/署名信息保留:"MIT · 可自由引用"是信息,不是装饰。

## 4. 状态变化的表演

- "✨ 太棒了!您已成功记录本条数据" → "已记录"
- "数据已成功复制到您的剪贴板,请前往目标位置粘贴" → "已复制"
- 空状态废话("当前没有任何数据,请先开始填写")→ 删,空状态留白。UI 本身就是陈述。

## 符号规则

结构符号留,情绪符号删。

- **留**:表格标记(✓ / ✗ / —)、流程箭头(→)、列表序号。删掉一个,表格或流程就乱——它们是结构。
- **删**:✨ 🎉 emoji、感叹号堆叠。删掉一个,什么都不坏——它们是装饰。

判据 = 删词测试的符号版:删掉这个符号,结构还读得通 → 装饰;塌了 → 结构。

# Forbidden — delete on sight

No test needed. These never survive.

## 1. Emotional symbols

- Emoji of any kind in professional/archival copy
- ✨ 🎉 and decoration glyphs
- Stacked exclamation marks (!!、!!!)

## 2. Gratitude / share-bait boilerplate

Delete the whole sentence:

- "感谢您耐心阅读本文,如有收获,欢迎分享"
- "如果觉得有用,请转发给身边的朋友"
- "欢迎点赞、收藏、关注"

## 3. Author puffery

Keep the name, delete the title stack:

- "本文作者X,资深独立研究者,长期深耕该领域多年" → "X"

Licensing/attribution stays if real: "MIT · 可自由引用" is information, not decoration.

## 4. Status-change theater

- "✨ 太棒了!您已成功记录本条数据" → "已记录"
- "数据已成功复制到您的剪贴板,请前往目标位置粘贴" → "已复制"
- Empty-state filler ("当前没有任何数据,请先开始填写") → delete; leave the empty state blank. The UI is the statement.

## The symbol rule

Structural symbols stay; emotional symbols go.

- **Stay**: table marks (✓ / ✗ / —), flow arrows (→), list numbering. Delete one and the table or flow breaks — they are structure.
- **Go**: ✨ 🎉 emoji, exclamation stacks. Delete one and nothing breaks — they are decoration.

Criterion = the delete-test, symbol edition: remove the symbol; if the structure still reads, it was decoration; if it collapses, it was structure.

# 删词测试

决定"删词"还是"删整句"的唯一判据。

## 步骤

1. 取一句。
2. 删光句中全部形容词、副词。
3. 读剩句。
   - **成立**(完整、有信息的陈述)→ 用剩句。
   - **残句**(语法碎片,或一句什么都没说的话)→ 整句是装饰 → 删整句。

## 删词与删整句的分界

问剩句:缺的是**信息**,还是只缺**修饰**?

- 只缺修饰 → 删对了,用剩句。
  - "我们精心设计了简洁的导出流程" → "导出流程"(作为标签)——成立。
- 缺信息,而且那信息本来就不在这句里 → 这句什么都没承载 → 删整句。
  - "读者将会深刻地感受到机制的精妙之处" → "读者会感受到机制"——一句什么都没说的话 → 删整句。

## 灰色地带

一遍判不准:把敬语助词(您/请/哦/呢)也删了,再判一次。两遍都活下来的是内容;活不下来的是包装。

## 测试不判断的东西

- 对错。克制的句子也可能是事实错误。
- 完整性。文本缺信息,删除补不上——报告缺口,不替写。

# visual.md — Visual tokens (reused from read-not-decorate)

This skill does not own a visual system. It reuses the restraint tokens
from the sibling skill read-not-decorate, so the collection stays visually
coherent. Listed here for standalone use.

## Tokens

- Background: `#FAFAF7` (warm off-white, not pure white)
- Body text: `#1A1A1A`; secondary: `#666666`; muted/meta: `#999999`
- Divider: `#E5E5E0`
- Accent: `#C8553D` — only for: section numbers, the one key judgment per
  page, caliber-caption labels. Never for decoration.
- Fonts: system stack — `-apple-system, "PingFang SC", "Helvetica Neue",
  sans-serif`. No webfont CDN.
- Headings weight 600, slight negative letter-spacing.

## Prohibitions

No emoji. No gradients, shadows, glows, large border-radius. No decorative
icons or stock images. No AI-generated illustration. No second accent
color. No share/like elements.

## Slide-specific

- One judgment sentence per page may be large (28–36px) and accented;
  everything else stays quiet.
- Page footer: page number + section name, muted color, nothing else.
- Whitespace is the primary structure signal; do not fill it.

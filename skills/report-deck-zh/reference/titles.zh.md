# titles.zh.md — 对仗标题规则

对仗标题是中文正式汇报最显眼的标志。同一章节组内的同级标题必须同时
满足三条:

## 1. 字数对齐

组内标题字数一致。8 字是主力,6 字、10 字也常见。一组内长短混用
= 对仗破。

## 2. 结构对称

组内词性骨架一致。第一条是动宾+动宾,其余每条也是动宾+动宾。

## 3. 动宾一致

章节讲动作,统一动词开头;讲状态,统一名词开头。组内不许混。

## 生成流程

1. 先把每节要说的话写成大白话句子。
2. 从每句里抽出核心动词+宾语。
3. 以最强的一条定组模板(如 4+4:动宾+动宾),其余各条改写进模板。
4. 把整组竖着念一遍。破坏节奏的那条重写,不找借口。

## 虚构示例(通用商业,4+4 模板)

- 夯实供给基础,提升履约效率
- 优化获客结构,降低单客成本
- 收紧库存口径,加快周转节奏

反例(破掉的组):「夯实供给基础」「我们的获客做得更好了」「库存」
——长短混、结构混、形式混。

## 边界

正式汇报必须对仗。轻量内部分享强行对仗反而僵——该场景本规则不适用
(本 skill 也不适用)。

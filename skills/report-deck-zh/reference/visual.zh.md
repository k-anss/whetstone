# visual.zh.md — 视觉令牌(复用 read-not-decorate)

本 skill 不自带视觉系统,复用兄弟 skill read-not-decorate 的克制令牌,
保持合集视觉一致。此处列出以便单独使用。

## 令牌

- 底色:`#FAFAF7`(米白偏暖,不是纯白)
- 正文:`#1A1A1A`;次级:`#666666`;弱化/meta:`#999999`
- 分隔线:`#E5E5E0`
- 强调色:`#C8553D`——只用于:章节序号、每页那一句关键判断、口径
  说明的标签。绝不用于装饰。
- 字体:系统字体栈 `-apple-system, "PingFang SC", "Helvetica Neue",
  sans-serif`。不外链字体 CDN。
- 标题字重 600,轻微负字间距。

## 禁区

禁 emoji。禁渐变、阴影、发光、大圆角。禁装饰性图标与 stock 配图。
禁 AI 生成插画。禁第二种强调色。禁分享/点赞类元素。

## slides 专属

- 每页允许一句大字判断(28–36px)可用强调色;其余保持安静。
- 页脚:页码 + 章节名,弱化色,别的不放。
- 留白是首要的结构信号,不许填满。

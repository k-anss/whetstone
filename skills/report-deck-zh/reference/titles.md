# titles.md — Parallel (对仗) title rules

Parallel titles are the most visible marker of a formal Chinese briefing.
Same-level titles in one section group must satisfy all three:

## 1. Aligned character count

All titles in the group have the same number of Chinese characters.
8 characters is the workhorse; 6 and 10 also common. Mixed lengths in one
group = broken parallelism.

## 2. Symmetric grammatical structure

Same part-of-speech skeleton across the group. If title one is
verb-object + verb-object (动宾+动宾), every sibling is too.

## 3. Consistent verb-object form

Lead with a verb when the section reports action; lead with a noun when it
reports state. Never mix within a group.

## Generation procedure

1. Write each section's message as a plain sentence first.
2. Extract the core verb + object from each.
3. Pick the group's template (e.g., 4+4: 动宾 + 动宾) from the strongest
   title, then rewrite siblings into the template.
4. Read the group aloud as a column. Any title that breaks rhythm gets
   rewritten, not excused.

## Fictional examples (generic business, 4+4 template)

- 夯实供给基础,提升履约效率
- 优化获客结构,降低单客成本
- 收紧库存口径,加快周转节奏

Counter-example (broken group): 「夯实供给基础」「我们的获客做得更好了」
「库存」 — mixed length, mixed structure, mixed form.

## Boundary

Mandatory for formal briefings. For light internal shares, forced
parallelism reads stiff — this rule does not apply there (and neither does
this skill).

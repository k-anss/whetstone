# data-pages.zh.md — 数据页与口径说明

口径说明是中文正式数据页的专业标志。没有口径的数字,等于没有依据的
论断。

## 数据页的解剖

1. **页标题**——这页数据支撑的判断(与同级标题对仗)。
2. **数字本体**——表格或简单图。一页一个信息。
3. **口径说明(强制)**——页底小字,三格,全填:
   - 数据来源:数字出自哪个系统/台账/报表。
   - 统计范围:含什么、不含什么。
   - 时间口径:精确时段,以及是时点值还是累计值。

哪格不掌握,就在那格明写(如"统计范围:待补")——缺失本身是信息,
不许静默丢掉。"不掌握"和"不适用"是两种状态,不许压成一种。

## 排版规则

- 口径:更小字号、次级颜色、不装饰、不省略。
- 表格和花哨图都能用时,选表格。禁 3D、禁渐变。
- 数字右对齐;单位在表头写一次,不逐格写。
- 对比数字在口径里写明对比基准(同比/环比的具体基期)。

## 虚构口径示例

> 数据来源:订单系统导出;统计范围:自营渠道,不含第三方代销;
> 时间口径:2025-07-01 至 2025-09-30,累计值。

## 边界

本式样针对汇报中作为证据呈现的数字。探索性分析笔记是另一回事,
不在范围内。

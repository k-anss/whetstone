# 视觉系统

硬约束。不是建议;偏离这些约束,正是本 skill 要防止的失败模式。

## 颜色

| 角色 | 值 |
|---|---|
| 底色 | `#FAFAF7`(米白偏暖,绝不用纯白) |
| 正文字色 | `#1A1A1A` |
| 次级字色 | `#666666` |
| 弱化字色(meta、附注) | `#999999` |
| 分隔线 | `#E5E5E0` |
| 强调色 | `#C8553D` |

强调色是全文档唯一的颜色。它**只**出现在:关键判断、章节序号、自案例边框、
引用块竖线、目录序号、进度条。其余任何地方不上色。

禁用:任何第二种颜色、蓝、绿、渐变、投影、发光、大圆角卡片。

## 字体

只用系统字体栈。不用 web 字体,不走 CDN。

- 正文/标题:`-apple-system, "PingFang SC", "Helvetica Neue", sans-serif`
- 引用块:`Georgia, "Songti SC", serif`,斜体

标题字重 600,字间距 H1 为 `-0.02em`、H2 为 `-0.015em`。

桌面端字号节奏:

| 元素 | 字号 / 行高 |
|---|---|
| H1 | 42px / 1.2 |
| H2 | 28px / 1.3 |
| H3 | 20px |
| 正文 | 17px / 1.75 |
| 引用 | 17px 斜体 |
| 关键判断 | 22px,字重 600 |
| 附注/次级 | 13–15px,`#666` 或 `#999` |

## 间距

- 章节之间:≥ 80px
- 段落之间:1.4em
- 关键判断前后:各 60px
- H2 的 `scroll-margin-top`:80px(锚点跳转不被 sticky 导航遮挡)

## 版心

正文最大宽度 **680px**,居中。绝不满屏铺开。

## 页面骨架(每篇文档都有)

```
[顶部 sticky 导航,浅毛玻璃]
  左:标题简写(留档版加一个强调色"留档"小字标)
  右:目录 ▾(点开抽屉)

[进度条,顶部 2px 细线,强调色,随滚动更新]

[目录抽屉,从右侧滑出]
  H2 列表,序号 01/02 用强调色,点击锚点跳转

[文章主体]
  Hero 区:H1 / 一句话定位(次级字色) / meta 行:作者 · 日期 · 字数(弱化字色)
  ───
  Section 01 …
  Section 02 …
  ───

[页脚签名档]
```

CSS 预算:总量不超过 600 行。超了,说明你在装饰。

# Copy Restraint

This governs **only the text you generate** — contents labels, component LABELs,
fold summaries, buttons, the signature block. **You do not touch one word of the
author's body text.**

## Three rules

1. Verb-driven. No adjective boosting.
2. Do not explain what you did.
3. Do not draw the reader's conclusion for them.

## Examples (LABEL / summary / button)

| Don't | Do |
|---|---|
| ✨ Great! You expanded this section | (expanding a fold needs no message) |
| Click below to view a detailed boundary analysis | Boundary and exceptions |
| A quick overview of the key points to help you grasp the whole | Contents |
| Below is the author's carefully crafted case study | First-hand case |
| Offering a comprehensive, multi-angle comparative view | (write nothing, show the contrast block) |
| Key judgment: this is critically important, pay attention! | (the judgment itself is enough) |

## Examples (signature / meta)

| Don't | Do |
|---|---|
| Thanks for reading; share if you found it useful | (delete the whole line) |
| By the author, a seasoned independent researcher who… | (the name) |
| Original work, all rights reserved, attribution required for reuse | (the actual license line) |

## The grey-area test

Unsure? Delete every adjective and adverb in the sentence and see if it still
stands. If it stands, use the stripped version. If it does not, the sentence was
decoration — delete it whole.

> "The reader will profoundly feel the elegance of the mechanism" → strip → "the
> reader will feel the mechanism" → fragment, does not stand → **delete whole.**

## Forbidden everywhere

No emoji. No animal metaphors (lead sheep / wolf / herd). No share/like/follow
furniture. No reading-time estimate. No long author bio. No stock images. No
AI-generated illustration (any SVG you draw is thin and linear). No "if you
found this useful" closers. The accent color never lands on decorative elements
— backgrounds, buttons, borders are all greyscale; only judgments, cases,
numbers, and rules carry the accent.

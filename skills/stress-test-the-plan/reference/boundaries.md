# Boundaries

This file is the skill's actual increment. The interrogation moves in
three-holes.md are not scarce; the discipline below is.

## When not to enter

**Emotional distress.** Signals: the user is venting, describes a
setback in personal terms, asks for reassurance, or mixes the plan with
how they feel. Action: do not enter the role. Say plainly: "this is not
the moment for a stress test — we can do it later." Do not soften by
running a "gentler" interrogation; a gentle interrogation is still an
interrogation.

**Already decided, already executing.** Signals: "this is live",
"we shipped", "I've committed", asking for optimization not evaluation.
Action: do not interrogate "should you do it" — that question is closed
and reopening it is interference. Narrow the scope to "how to die less":
which execution risk kills fastest, what to verify next, where the
plan's own assumptions can still bite during execution.

**Ambiguity.** If you cannot tell whether the user wants challenge or
support, ask one question before entering. Entering wrong is worse than
asking.

## How to exit

The user says stop — any phrasing: "enough", "stop the stress test",
"let's change topic". Exit immediately and completely:

- Drop the adversarial stance in the same turn.
- Do not append "one last point".
- Do not carry the pessimistic tone into the next topic.

A role that lingers after dismissal has become the master, not the tool.

## Declared overcorrection

This role is deliberately pessimistic. It will, by construction,
sometimes talk a good opportunity out of existence. Two obligations
follow:

1. **Every run ends with the midpoint statement**: "This is the
   deliberately pessimistic extreme. Take the real judgment between
   this and a normal read." Mandatory output, not optional tone.
2. **Never present the verdict as the final word.** The role's verdict
   is one pole of the read. The user owns the midpoint.

## What the role still must not do

- No personal attacks. Assume the user is smart with blind spots,
  motivated but framed by their own story — not stupid, not lazy.
- No comfort disguised as judgment ("but that doesn't mean you can't
  do it"). Give the real read, then the midpoint statement.
